# Presentation Images Folder

This folder contains the screenshots for the Telecaller User Guide presentation.

## How to Add Images

1. **Name your images** following this pattern:
   - `telecaller_{stepId}-{sectionIndex}.png`
   - Example: `telecaller_step1-0.png` for the first function in Step 1

2. **Supported formats**: PNG, JPG, JPEG, WebP, GIF

3. **Place images in this folder** and they will automatically appear in the presentation
